# pic-writing-board

PIC writing board for DIP package with auto guess pin count.

## Core Components

|            |                   |
| ---------- | ----------------- |
| Analog MUX | TMUX1309BQBR      |
| MCU        | PIC18F27Q10T-I/ML |
| Comparator | LM2901Q4T         |
| SSR        | TLP172AM          |
| LDO        | NJM2845           |
| Power SW   | TCP8407           |
